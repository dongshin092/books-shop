
# 1. 장바구니 기능 구현
-  도서 상세 화면, 메뉴별 도서리스트 에서 장바구니 버튼을 누를 경우  기능 구현
-  장바구니에 도서가 등록되면, 장바구니로 갈지, 아니면 계속 쇼핑할지
   사용자에게 문의하는 팝업을 노출 
- 장바구니 이동을 누르면 장바구니 페이지로 이동
- 계속 쇼핑을 누르면 팝업해제 후 해당페이지에 유지 
- 장바구니는 로그인 시 사용가능한 기능
- 비 로그인 시 선택할 경우, 로그인이 필요한 기능입니다, 로그인하시겠습니까? 라고 문의하고
  OK를 누르면 로그인페이지로 이동, NO 하면 팝업 제거하고 페이지 유지 


# 2. API 연동
- 장바구니 추가 API 는 서버에 이미 구현 되어 있음
- API 설명
    - POST 방식
    - 매개변수: {"bookId" :"책ID", "quantity" : "1" }
    - 수량은 1 로 고정
    - end-point : /api/carts
    - userId는 현재 로그인한 사용자 Id를 입력
    - 만약 동일 bookId가 존재하면  수량만 1 증가하여 업데이트
    - 동일 bookId가 없다면  insert
